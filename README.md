# SpendPilot AI Audit

> AI-powered SaaS spend audit tool — built for the Credex 7-day engineering challenge.

## Overview

SpendPilot AI Audit helps companies identify wasteful SaaS subscriptions, unused licenses, and cost optimization opportunities using AI-driven analysis.

## Stack

- **Frontend:** React + Vite + TailwindCSS + Framer Motion
- **Backend:** Node.js + Express + MongoDB (Mongoose)
- **AI:** OpenAI API
- **Email:** Resend

## Getting Started

### Frontend
```bash
cd client
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```
